# Dirb-like



simple program, made to be a copy of Dirb program, written in native python. 

 BASIC RULES:

   [ ] - NO Ai help. otherwise this would take 10 minutos to be done
   [ ] - NO external libs, meaning no pip install. 
   [ ] - Must use threading or multiprocess (Pool)
   [ ] - Modular Wordlist
   [ ] - Output in JSON
   [ ] - Possibility of exploring in depth of a 200 url 

USAGE: 

    python dirbpy.py -w [/wordlist.txt] --args
